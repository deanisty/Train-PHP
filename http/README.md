### Hypertext Transfer Protocol (HTTP) 超文本传输协议

>
> 参考： https://developer.mozilla.org/en-US/docs/Web/HTTP
>

* HTTP是一个应用层协议
* 它被用作传输超媒体文档，例如 HTML。
* HTTP 协议基于经典的 CS 通信模式，客户端打开一个连接请求，等待服务器的响应。
* HTTP 是一个无状态协议，它不会在两次请求之间保存任何数据。
